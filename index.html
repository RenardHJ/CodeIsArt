PP: Abstract Code Visualizer

Python Portraits

Python Portraits is a web-based abstract image generator. It analyzes the lines of a python file and converts the program structure into an abstract image. All of the image generation is done on your CPU, therefore the time it takes to create the image is dependant on your computer.

Python Portraits is still in pre-alpha. We are working hard to become feature-complete, however if you would like to report an issue or suggest a feature please contact us.
