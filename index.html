Code Is Art: Abstract Code Visualizer

Code Is Art

Code Is Art is a web-based abstract image generator. It analyzes the lines of a python file and converts the program structure into an abstract image. All of the image generation is done on your CPU, therefore the time it takes to create the image is dependant on your computer.

Step 1. Load a .py or .json file

Step 2. Choose your visualization Style

  • Barcode
  • Pixel Sorting
  • Fractals
