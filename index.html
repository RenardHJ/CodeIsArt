VIP: Abstract Code Visualizer

Visual Interpreter - Python

VIP is a web-based abstract image generator. VIP analyzes the lines of a python file and converts the program structure into an abstract image. All of the image generation is done on your CPU, therefore the time it takes to create the image is dependant on your computer.
